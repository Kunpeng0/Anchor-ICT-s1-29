# Anchor-ICT-s1-29-

Project Setup Guide

This project runs locally using Node.js and a development server. Follow the steps below to get started.

1. Install Node.js
Go to Node.js
Download the LTS (Long Term Support) version (recommended for stability)
Run the installer and proceed with the default settings

2. Open the Project

Navigate to your project folder and open it in your code editor:

code .
3. Install Dependencies

Install all required project packages:

npm install
4. Start the Development Server

Run the following command to start the local development environment:

npm run dev
5. Open in Browser

Once the server is running, open your browser and go to:

http://localhost:5173
Notes
Ensure Node.js is properly installed before running commands
If code . does not work, make sure your editor is added to your system PATH
The development server will automatically reload when changes are made
